Scroll down and add a commit message, then click "Commit new file". Step 4: Enable GitHub Pages Go to the Settings tab of your repository. Scroll down to the "Pages" section in the sidebar. Under "Source", select the branch you want to use (usually main) and the folder (typically / (root)). Click "Save". Step 5: Access Your Website After a few minutes, your website will be available at https://username.github.io (replace username with your GitHub username). You can visit this URL in your web browser to see your new website. Step 6: Make Changes To update your website: Go back to your repository. Click on the file you want to edit (e.g., index.html). Click the pencil icon to edit the file. Make your changes, add a commit message, and click "Commit changes". Step 7: Explore Further You can add more HTML files, CSS for styling, and JavaScript for interactivity. You can also use Jekyll, a static site generator, for more complex websites.
